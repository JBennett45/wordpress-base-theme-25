# WordPress Base Theme 2026 (Classic Theme) 
Starter WordPress classic theme based with support for ACF classes & vite preprocessing.

**Demo:** You can access a demo of begin state of the theme [here](http://basetheme.jbdev.co.uk).

**Note:** this isn't a block/gutenburg theme, its based on classic build style with ACF support (if installed)

# Installing Dependencies 
Run npm i to install the depencies, the npm run dev for asset watching, npm run build for production.

# Elements
There are various elements you can see on the demo site such as on page blog loading/filter, accordions and more. 

# Development Mode
Installed within the theme is the ability to put the theme in development mode, which is managed by a 'master' username. When the master name is set and status is set to development, anyone other than the master username will be signed out and shown a message about the backend being in development. To reverse simply login in as master and change the status to staging or production.
